Shop Imports via Xentral Connect

Xentral Connect is the new integration hub in Xentral.

Key features include:

  • Bidrectional Out-of-the-box Import / Export Workflows between Xentral and different Onlineshops and marketplaces
  • Build on our own node Integration platform as a service “Xentral Connect”
  • Standard-Integrations for
    • Shopify
    • WooCommerce
    • Shopware 6
    • BigCommerce
    • Otto
    • Mirakl
    • Tradebyte
  • More Integrations based on legacy technology
    • eBay
    • Amazon
    • Kaufland
    • Magento2
    • Gambio
    • Presta
    • etc.
  • All Integrations offer Order2Cash functions which means
    • Sales Order Import
    • Stock Sync
    • Tracking information export
    • Order state sync
    • Invoice export if possible
  • Major integrations also offer product data sync
    • Article basedata
    • Media
    • Properties
    • Freefields
    • Prices
    • …
  • Connect workflows can be monitored and managed in Xentral UI
  • All Out-of-the-box workflows can be customized using Connect No-Code Engine
  • All integrations can be setup and configured directly in Xentral UI
  • Import/Export Workflows can be continuously monitored using the integrated journal function. Furthermore Connect workflows are directly integrated with System Health Page
